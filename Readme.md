## 🛠️ How To Install (Manual)

1. Go to `chrome://extensions/`
2. Enable **Developer Mode** (top right)
3. Click **"Load Unpacked"**
4. Select your folder (e.g., `~/t00ls/JS4hacking/`)
5. Boom! Now hit any target site, open DevTools, and use the extension!

---

## 🧩 Using the Extension
- Click the extension icon in Chrome toolbar
- Toggle ON/OFF for tool usage
- Click buttons (e.g., `Links`, `Mails`) to inject respective scripts
- You can always write your own in `Scripts/` and hook them in
